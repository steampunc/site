Instructions

Input cards by specifying their four attributes in a string:

  • Quantity (1, 2, or 3)
  • Shape (o, d, or s) for Oval, Diamond, and Squiggle respectively
  • Color (r, g, or p) for Red, Green, or Purple
  • Fill (f, h, e) for Full, Hatched, or Empty

These four attributes can be entered in any order (so 2rof is identical to r2fo or any other permutation of the 4 attributes). Select between sets by using the arrow buttons.
